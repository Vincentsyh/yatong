永康市亚彤工贸有限公司 ——

专业制做、信誉、高质量保证十年载

       永康市亚彤工贸有限公司是一家模具设计、模具制作、注塑加工、移动电源外壳及PCBA制造、移动电源设计生产、销售集合为一体的技术企业。成立于2010年2月，凭借高信誉、高质量、高效率，在同行业中迅速崛起。 依靠科技球发展，不断的向客户提供高科技产品，是我们始终不变的追求。
       公司在发展过程中，不断的吸取着各同行的交流，合作，对于客户的建议，我们也积极的听取和采纳，使得我们销售领域越来越广泛，并且不断的改进产品。我们的宗旨是使每一位顾客满意，我们以不断的努力和追求，制作出各种各样的产品，从而使得顾客一致好评。 我们的员工奉行“进取、求实、严谨、团结”的方针，不断的开拓创新，以技术为核心，视质量为生命、奉顾客为上帝，竭诚为您提供性价比高的科技产品、高质量的工程设计以及无微不至的售后服务。
